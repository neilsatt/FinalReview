A semantic element clearly describes its meaning header: A container to be used for a web page header which typically contains the site logo, heading elements, and site navigation. footer: A container to be used for a web page footer which typically contains contact, and copyright and nav links main: A high-level element used to contain all of the content that is unique to a single web page and not repeated across multiple web pages. nav: An element to contain blocks of site navigation links. aside: identifies content that is related to the main content on the page but not part of the primary flow of the document article: identifies a block of content suitable for reuse and syndication in other settings
